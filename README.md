## About

**SEDIMENT** is a slow-burn incremental game where you excavate geological layers, uncover rare artifacts, and build an archive of Earth's hidden history.

The deeper you dig, the older — and stranger — the world becomes.

### Core Loop
1. **Dig** — reveal tiles in a grid, uncover finds ranging from common stones to ultra-rare shiny specimens
2. **Archive** — display your finds to generate passive income and unlock story fragments
3. **Automate** — invest in tools, robots, and scanners to free yourself from manual labour
4. **Prestige** — break through to deeper geological eras, each with new aesthetics and secrets

### Layers
| Layer | Era | Atmosphere |
|---|---|---|
| 1 | Holocene (0–5m) | Warm, earthy |
| 2 | Pleistocene (5–30m) | Cool, blue-grey |
| 3 | Mesozoic (deep) | Dark, amber |
| 4 | Paleozoic (very deep) | Violet, mystic |
| 5 | Earth's Core (prestige) | Red, molten |
